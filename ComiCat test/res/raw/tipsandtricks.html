Tips to useful use cases

Hide one or more comic series from my kids

There are two ways you can protect you Comic collection from unwanted access.

  • Password Protecting ComiCat: By setting an app password you can protect your entire comic collection. Password can be set from "General Settings"
  • Using Private Folders: You can selectively choose to mark certain folders as private. Private folders and their comics can only be seen by entering the password.

Keep the zoom fixed while reading a comic

Sometimes it is very desirable to have a fixed zoom level for the entire comic, fortunately one of our user suggested this feature and we implemented it.

  1. Open the Comic
  2. Pinch to zoom to select the desired zoom level
  3. Pan the page by dragging to set the starting position for all subsequent pages
  4. Bring up the menu and select the icon with four inward arrows to lock zoom and page position.
  5. Continue enjoying your comics.

White borders around my comic are taking a lot of screen space

There are two ways in ComiCat to fix it

  • In the "Viewer settings" you can select, "Crop Comic Page Margins" to automatically crop the white borders from all the pages while displaying them.
  • While reading the comic, set the zoom level and position so that white borders are off screen. Now you can select "Set frame" option (the icon has four arrows pointing inwards) from the menu/action bar to lock the zoom level and position. All forthcoming pages during that reading session will be displayed with the same zoom level and position.

My comic collection is spread across multiple sd cards and USB drives

You can use the "Retain Deleted Comic Data" feature. This is a setting in "Catalog Setting", by default it is off, you can turn it on. Now, when you remove a SD card and insert another the resync will as usual add new comics from that SD card and will remove the ones from the one that was removed. But it will not delete the comic data, thumbnails, etc. So when the SD card is re-inserted the sync will be very fast and reading history, bookmarks, hidden state of comics will be maintained.

Remove a folder from my bookshelf without deleting

  1. In "Catalog Settings", select "Group Comics by Folders".
  2. Go to catalog and locate the folder you want to remove.
  3. Tap and hold to bring up the context menu.
  4. Select "Exclude Folder From Catalog"
If you ever wish to re-add the folder to catalog you can do so by going to "Catalog Settings" and removing the folder from the list of "Exclusions". You would need to re-sync after that.

Select the files that I want to download.

In the "Cloud Sync Settings" turn on "Enqueue Comic As Paused". Next time you do Cloud Sync all the new comics will be added to your download queue but as paused. You can start the desired ones from "Active Downloads". You can also set the priority for the comics you want first or choose to ignore a file so that it is never downloaded.

Read the entire series as a single comic

  1. In "Catalog Settings", select "Group Comics by Folders".
  2. Go to catalog and locate the folder that contains your series.
  3. Tap and hold to bring up the context menu.
  4. Select "Start Reading"
