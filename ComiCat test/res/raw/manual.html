Settings and options for ComiCat

General Settings

Start application in

Controls whether the app will be re-launched in catalog view or the comic that your were reading when you closed the application.

Open comics from

When opened, a comic can be set to open from:

  • Last read page: This is the last page you read before closing the comics
  • Bookmark: It works like a traditional bookmark. You can place a manual bookmark in the comic and then open it from there. Alternatively, you can open a comic from bookmark through the context menu in catalog view.
  • Start: Open from start regardless of where you left the comic or placed a bookmark.

Password protect

You can protect your comic collection from unwanted access by password protecting ComiCat. You will be asked to choose a numerical code as your password. You would need to enter that code to unlock the application on each re-launch.  There is a grace period of around 8 seconds within which you can re-launch the app without password prompt.  Please choose a password that you can easily remember as there is no way to recover a lost password. You would need to re-install the app if you forget the password. Unchecking this will disable the password protection.

Viewer Settings

Orientation

This option controls the viewer layout with respect to device orientation:

  • Sensor (Default): Viewer window responds to device rotation and sets itself in portrait or landscape mode depending on the device orientation.
  • Portrait: Lock the layout in portrait mode and do not respond to device rotation.
  • Landscape: Lock the layout in landscape mode and do not respond to device rotation.

View mode

This option controls how comic pages fit inside the viewer window:

  • Fit Visible: Resize the comic page so that height and width are contained within the viewable region. Page aspect ratio is maintained.
  • Fill Visible: Stretch the comic page to occupy all available space on screen. Page aspect ratio is not maintained.
  • Fit Width: Resize the comic page so that the width matches the width of the viewing region. Page can be scrolled vertically if the scaled height is more than the height of viewing region.
  • Fit Height: Resize the comic page so that the height matches the height of the viewing region. Page can be scrolled horizontally if the scaled width is more than the width of viewing region.
  • Original Size: Do no resize the page and show it in the actual dimensions. Page can be scrolled in any direction if the page dimensions are larger than the dimensions of the viewing region.

Page Transitions

The animation that is displayed when you move from one page to another. Options are

  • Page Curl
  • Page Shift
  • Page Slide

Fit width on rotation

When you rotate your device to landscape mode this option automatically changes the view mode to 'Fit width' for best reading experience.

Crop comic page margins

Some comics have white margins around the pages. On devices this dead space takes up otherwise useful screen real estate.  Turning on this option will detect and crop the white margins on the fly to give you more viewing space for your panels. Please note, only the extracted image is cropped, ComiCat doesn't modify the comic file itself.

Image Enhancer

A tool for improving scans of olden comics. It cleans up image, enhances color, brightness and removes paleness.

Double Page Spreads

Some comic files have all or some pages as double page spreads. Reading such comics on devices with limited screen space is not a very pleasant experience. This preference determines what ComiCat shall do in this case.

  • Split to single page spreads: Double page spreads will be split into single page spreads for display.
  • Split only in Portrait mode: Split into single page spreads only when the device orientation is portrait.
  • Do Nothing: Don't split.

Always hide titlebar

Keeps the top bar containing title and page number hidden by default. This gives you more reading space. You can also do the same temporarily using 'Hide Title' menu item in the Viewer's options menu. That change lasts only for that reading session.

Right to left reading

Traditionally, manga stories flow from top to bottom and from right to left. This option configures the reader for reading unflipped manga comics.

Limit gesture zone

Sets the screen area where tap, fling, swipe and double tap gestures will work. Pinch to zoom and panning always works across the screen. Options are:

  • Don't Limit (default): Gestures will work on entire screen area.
  • Bottom 25 percent: Gestures will only work on bottom 25 percent of the screen.
  • Top 25 percent: Gestures will only work on top 25 percent of the screen.

Disable swipe on zoom

When reading a comic in zoom mode, one often pans and flings the panel to move across page regions. Sometimes these gestures are recognized as swipe gesture and page is changed in response.  If this is bothering you, you can turn this option 'ON' to disable swipe gesture for page change. You can change the page by tapping on the screen.

Brightness level

By default the brightness level of the screen is managed by device preferences. You can however, set a fixed brightness level for the Viewer window using this option.

  1. Tap on the option to bring up the 'Brightness Picker' dialog.
  2. In the dialog, select 'Manage Brightness Level' to turn this on. This will also enable the slider.
  3. Move the slider by dragging, screen brightness will change in response.
  4. Dismiss the dialog once you have set the slider to desired brightness level.
  5. To turn off, deselect 'Manage Brightness Level' option in the dialog.

Use volume Controls

Turning this on allows you to use volume controls for page navigation.

Swipe Senstivity

When set to high, even a slight swipe will be recognized as a page change swipe gesture while a more explicit swipe will be required in low senstivity mode.

Viewer menu Options

Settings

Brings up the settings screen.

Bookmark

Sets a bookmark at the currently opened page, this bookmark can later be used to open the comic directly from this page.

Go To

Displays a scrollable list of page thumbs at the bottom of the screen, you can tap on a page thumb to quickly jump to it.

Set Frame

When this option is on all new pages are presented with the same zoom level and scroll position as it was when frame was set. This can be used for setting manual zoom level, setting initial page position or both.

Touch Options

Brings up a dialog where you can turn certain gestures on and off. Options are

  • Swipe for page turn: Enable/disable swipe gesture for page navigation.
  • Tap for page turn: Enable/disable tap gesture for page navigation.
  • Double tap to toggle view mode: Enable/disable double tap gesture for cycling through the available page view modes.
  • Longpress at bottom for fast seek: When enabled, tap and hold at the bottom right will fast forward and bottom left will revind the comic.
  • Disable swipe on zoom: If in zoom mode your page panning gestures are frequently being recongnized as page change then you should turn this option on. This will disable swipe gesture as soon as you use pinch to zoom on a page.
  • Tap and hold - Menu for page options: Tap and hold will bring up the menu and action bar.
  • Left edge swipe - Settings: Enables/disables the gesture for bringing up the settings drawer.

Hide Title

Hides the title and page number that is displayed at the top of the page. The change only remains in effect for one reading session.

Set Wallpaper

Sets the current page image as the device background. This option is not available on devices that don't support setting wallpapers.

Restart

Restarts the comic from page one.

Catalog Settings

Resync Catalog

Triggers a re-scan where it reconciles the changes made to the on-disk comic collection. It will scan for new comics and deletions. Old comics and their reading history, bookmarks, visibility etc will be preserved.

Include Zip, Rar, PDF format files

  • Always Include: All the zip, rar, 7zip, pdf extension files will be scanned and added to your catalog.
  • Include only if inside comic folder: Only the files that are inside a folder containing "comic" in its name or also have cbz, cbr files will be included.
  • Don't Include: Files with these format extensions will not be included.

Rescan on start

Convenience function to do a re-scan every time the app launches. This keeps your catalog up-to-date without the need for manual resync. Turn this on if you frequently make catalog changes.

Fix file extensions

Some comic files have the wrong extension. for example cbz format files will have cbr extension and vice-versa. These files can be opened by ComiCat but they take a little longer than what they will with the right extension. Check this option to check and fix the extensions while scanning.

Exclusions

You can see and manage what all files and folders you have chosen to exclude from you catalog.

Group Comics By Folders

Selecting this option creates a 2 level catalog shelf, on the first level there are your folders and selecting the folder will show the comics in that folder. Please note, this does not replicate the hierarchy you have on your storage, it displays all the folders at one level.

Sort by

Default sort order for catalog display. This can also be changed from catalog options menu.

Catalog Theme

ComiCat comes with multiple catalog themes. The default is the wooden theme. You can also try Steel Mesh, Titanium and Cool Blue and Black Wood themes. If you want the plain vanilla look, please choose the Black theme.

Display Large Thumbnails

Selecting this will enable large thumbnail display on your shelf, particularly useful for big screen devices.

Clear bookmarks on finish

Once you have finished reading the comics and have marked it read, setting this option to 'ON' will automatically clear any bookmark that you might have placed.

Retain Deleted Comic Data

When this option is set the information for deleted comics is not removed from the database. So when the same comic is added later, all its reading history, bookmarks, hidden state etc is retained. This is helpful for people having multiple SD cards or USB drives containing their comic collection.

Performance Settings

Max image size

Every app has a max memory limit that it can use for its working.  If the comic page image is too large it can cause the app to exceed that limit and the app may crash.  For the stability of ComiCat we keep this maximum limit and any image beyond this limit is resized to lower resolution.  Default value is good unless you are getting crashes while viewing comics.

Show transitions

Controls whether page change animation shall be displayed. Disabling transitions reduces the amount of images ComiCat needs to cache at any given point.  By default it is enabled. Turn off only if you are getting crashes or you don't want to see the transitions at all.

Aggressive caching

Cache more objects for increased speed and performance. This option is not available on devices with lower memory limit.

Private Folders

Enable private folders

This option can be used to hide any set of comics that you don't want your kids or other people to see.  When you enable private folders you are asked to pick a numerical password that will be used to control any change to private folder settings and showing/hiding of private folders.  You need to pick a password that you can easily remember, there is no way to recover a lost password.

Select private folders

Select folders containing comics that you want to hide. When hidden, these folders will not be visible in the folder selector drop down. Any comics in private folders will not show up in the catalog or search results unless the folders are set to visible.

Hide on re-launch

Controls whether the private folders will be automatically hidden on application re-launch.  It's safer as you would not need to remember to hide the folders each time you give your device to somebody who shouldn't see the private folders and the comics in them.

Currently Hidden

Makes private folders visible or invisible. If you have set 'Hide on re-launch' to false than you would need to manually hide the folders by selecting this option.

Facebook Settings

Authorize ComiCat

This is where you authorize ComiCat to post comments and messages to your Facebook feed on your behalf.

Post Update when I start a Comic

This will post a message to your Facebook status when you start reading a new comic book. Network connection is required for this operation to be successful.

Post Update when I finish a Comic

When you are done reading a comic this will update your Facebook status.

Post Images

Selecting this option will post low resolution comic covers and page images along with statuses and your comments.

Cloud Sync Settings

Include Zip, Rar, PDF format files

  • Always Include: All the zip, rar, 7zip, pdf extension files will be scanned and downloaded.
  • Include only if inside comic folder: Only the files that are inside a folder containing "comic" in its pathname.
  • Don't Include: Files with these format extensions will not be included.

Maximum Parallel Downloads

Maximum number of concurrent downloads. If you are on a high speed network, setting it to maximum will give you best results.

Enqueue Comics as Paused

New comics found on your network will be added to your download queue. If this option is selected then these will be added as paused. You can then review and select the ones you want to download and start the download by resuming them.

Use only Wifi for downloads

Turn this on if you don't want mobile data network to be used for downloads.

Don't Download while roaming

If data downloads on roaming costs you more then you can turn this option on to avoid accidental downloads while roaming.

Auto clear completed downloads

Remove downloads from the list as soon as they are completed.

Download Completed Notifications

  1. No Notification: No notification will be posted on system notification bar.
  2. Text Only: Text message will be posted to system notification bar.
  3. Text and Notification tone: Text message will be posted to system notification bar and the device will play the alert tone.

Maintain Download History

ComiCat will maintain a history of all files download from cloud storage services. This prevents ComiCat from re-downloading a comic file that has been deleted from the device after downloading. Turn this off if you want to force re-download all deleted comic files.
